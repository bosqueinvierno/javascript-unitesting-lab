Herramientas


  • Qunit

Qunit


  • Desarrollado por el equipo de jQuery
  • Todo jQuery está testeado con Qunit
  • Qunit está testeado con Qunit
  • No depende de jQuery
  • Está pensado desde cero para testear Javascript

Blanket


  • Code coverage
  • Funciona por instrumentación
  • Funciona con Qunit y con otros frameworks como Jazmine y Mocha

PhantomJs


  • Implementación de WebKit standalone (Headless browser)
  • Posee una API para controlar su comportamiento
  • Se comando con Javascript
  • Permite hacer capturas de pantalla, análisis de tráfico
  • Automatización
